## 一、模块概述

模版管理模块提供系统模版的增删改查、导入导出功能，用于配置和管理系统中的各类模版。

### 1.1 功能清单

| 功能 | 接口 | 方法 | 说明 |
|------|------|------|------|
| 列表查询 | `/system/template/list` | GET | 支持模糊查询、时间范围筛选 |
| 新增模版 | `/system/template/add` | POST | 模版名称唯一校验 |
| 编辑模版 | `/system/template/edit` | POST | 模版名称唯一校验（排除自身） |
| 删除模版 | `/system/template/remove/{ids}` | POST | 批量删除 |
| 导出模版 | `/system/template/export` | POST | 导出Excel |
| 导入模版 | `/system/template/import` | POST | 导入Excel，支持重复校验 |

## 四、核心业务逻辑

### 4.1 唯一性校验（人大金仓兼容）

**问题**：人大金仓数据库对 CASE 表达式类型检查严格，MyBatis-Plus 的条件参数写法会报错。

```java
// ❌ 不推荐（可能报错：CASE 的类型 boolean 和 text 不匹配）
wrapper.eq(SysTemplate::getTemplateName, bo.getTemplateName())
       .ne(ObjectUtil.isNotNull(bo.getId()), SysTemplate::getId, bo.getId());

// ✅ 推荐（用 if-else 替代条件参数）
LambdaQueryWrapper<SysTemplate> wrapper = new LambdaQueryWrapper<SysTemplate>()
        .eq(SysTemplate::getTemplateName, bo.getTemplateName());
if (ObjectUtil.isNotNull(bo.getId())) {
    wrapper.ne(SysTemplate::getId, bo.getId());
}
```

### 4.2 查询条件构建

```java
private LambdaQueryWrapper<SysTemplate> buildQueryWrapper(SysTemplateBo bo) {
    Map<String, Object> params = bo.getParams();
    LambdaQueryWrapper<SysTemplate> wrapper = new LambdaQueryWrapper<>();
    wrapper.like(StringUtils.isNotBlank(bo.getTemplateName()), SysTemplate::getTemplateName, bo.getTemplateName())
            .eq(StringUtils.isNotBlank(bo.getTemplateType()), SysTemplate::getTemplateType, bo.getTemplateType())
            .eq(StringUtils.isNotBlank(bo.getStatus()), SysTemplate::getStatus, bo.getStatus())
            .between(params.get("beginTime") != null && params.get("endTime") != null,
                    SysTemplate::getCreateTime, params.get("beginTime"), params.get("endTime"))
            .orderByDesc(SysTemplate::getCreateTime);
    return wrapper;
}
```

### 4.3 导入逻辑

```java
@Override
@Transactional(rollbackFor = Exception.class)
public void importTemplate(MultipartFile file) {
    // 1. 读取Excel
    TemplateReaderListener listener = new TemplateReaderListener();
    try (InputStream inputStream = file.getInputStream()) {
        EasyExcel.read(inputStream, SysTemplateReaderDTO.class, listener)
                .sheet()
                .doRead();
    } catch (Exception e) {
        throw new BizException("Excel解析失败：" + e.getMessage());
    }

    // 2. 获取数据
    List<SysTemplateReaderDTO> list = listener.getList();
    if (CollectionUtils.isEmpty(list)) {
        throw new BizException("导入数据不能为空");
    }

    // 3. 查询已存在的模版名称（性能优化：一次性查询）
    List<SysTemplate> existingTemplates = baseMapper.selectList(new LambdaQueryWrapper<SysTemplate>()
            .select(SysTemplate::getTemplateName));

    Set<String> existingNames = existingTemplates.stream()
            .map(SysTemplate::getTemplateName)
            .filter(Objects::nonNull)
            .collect(Collectors.toSet());

    // 4. 校验用临时集合（Excel内重复校验）
    Set<String> excelNames = new HashSet<>();
    List<String> errorMsgList = new ArrayList<>();

    // 5. 遍历处理
    for (int i = 0; i < list.size(); i++) {
        SysTemplateReaderDTO dto = list.get(i);
        int rowNum = i + 2;  // Excel行号从第2行开始

        List<String> rowErrors = new ArrayList<>();

        // 校验模版名称
        if (StringUtils.isBlank(dto.getTemplateName())) {
            rowErrors.add("模版名称不能为空");
        } else if (existingNames.contains(dto.getTemplateName())) {
            rowErrors.add("模版名称已存在");
        } else if (!excelNames.add(dto.getTemplateName())) {
            rowErrors.add("Excel内模版名称重复");
        }

        if (!rowErrors.isEmpty()) {
            errorMsgList.add(String.format("第%d行【%s】：%s",
                    rowNum, dto.getTemplateName(), String.join("; ", rowErrors)));
        } else {
            // 插入数据
            SysTemplate entity = new SysTemplate();
            entity.setTemplateName(dto.getTemplateName());
            entity.setTemplateType(dto.getTemplateType());
            entity.setTemplateContent(dto.getTemplateContent());
            entity.setStatus(convertStatus(dto.getStatus()));
            entity.setRemark(dto.getRemark());
            save(entity);

            // 添加到已存在集合（防止后续行重复）
            existingNames.add(dto.getTemplateName());
        }
    }

    // 6. 统一抛出错误
    if (!errorMsgList.isEmpty()) {
        throw new BizException("导入失败：" + String.join("", errorMsgList));
    }
}
```

### 4.4 状态值转换

```java
/**
 * 状态值转换
 * 支持：0/1、正常/停用、0正常/1停用 等格式
 */
private String convertStatus(String status) {
    if (StringUtils.isBlank(status)) {
        return "0";
    }
    status = status.trim();
    if ("0".equals(status) || status.startsWith("0") || "正常".equals(status)) {
        return "0";
    }
    if ("1".equals(status) || status.startsWith("1") || "停用".equals(status)) {
        return "1";
    }
    return "0";
}
```

## 五、Controller接口

```java
@RequestMapping("/system/template")
@RestController
@Slf4j
@Tag(name = "模版管理")
@Validated
public class SysTemplateController extends BaseController {

    @Autowired
    private ISysTemplateService templateService;

    /**
     * 模版查询
     */
    @GetMapping("/list")
    @Operation(summary = "模版查询")
    public R<List<SysTemplateVo>> list(SysTemplateBo bo) {
        List<SysTemplateVo> list = templateService.selectTemplateList(bo);
        return R.ok(list);
    }

    /**
     * 新增模版
     */
    @PostMapping(value = "/add")
    @Operation(summary = "新增模版")
    public R<Void> add(@Validated @RequestBody SysTemplateBo bo) {
        if (!templateService.checkTemplateNameUnique(bo)) {
            return R.fail("新增模版'" + bo.getTemplateName() + "'失败，模版名称已存在");
        }
        return toAjax(templateService.insertTemplate(bo));
    }

    /**
     * 编辑模版
     */
    @PostMapping(value = "/edit")
    @Operation(summary = "编辑模版")
    public R<Void> edit(@Validated @RequestBody SysTemplateBo bo) {
        if (!templateService.checkTemplateNameUnique(bo)) {
            return R.fail("修改模版'" + bo.getTemplateName() + "'失败，模版名称已存在");
        }
        return toAjax(templateService.updateTemplate(bo));
    }

    /**
     * 删除模版
     */
    @PostMapping("/remove/{ids}")
    @Operation(summary = "删除模版")
    public R<Void> remove(@PathVariable String[] ids) {
        return toAjax(templateService.deleteTemplateByIds(Arrays.asList(ids)));
    }

    /**
     * 导出模版
     */
    @PostMapping("/export")
    @Operation(summary = "导出模版")
    public void export(SysTemplateBo bo, HttpServletResponse response) {
        templateService.exportTemplate(bo, response);
    }

    /**
     * 导入模版
     */
    @PostMapping(value = "/import", consumes = MediaType.MULTIPART_FORM_DATA_VALUE)
    @Operation(summary = "导入模版")
    public R<Void> importExcel(@RequestPart("file") MultipartFile file) {
        templateService.importTemplate(file);
        return R.ok();
    }
}
```

## 六、设计要点

### 6.1 性能优化

| 优化点 | 说明 |
|--------|------|
| 一次性查询已存在数据 | 避免N+1查询，用Set进行O(1)查找 |
| 使用Set集合 | `contains()` 和 `add()` 都是O(1)时间复杂度 |
| 插入后更新Set | 防止Excel内后面的行与前面行重复 |

### 6.2 用户体验

| 设计点 | 说明 |
|--------|------|
| 收集所有错误后统一抛出 | 用户可一次性看到所有问题，不用反复修改 |
| 友好错误提示 | 格式：`第X行【模版名称】：错误信息` |
| 状态值兼容 | 支持多种格式输入（0/1、正常/停用等） |

### 6.3 人大金仓兼容

| 问题 | 解决方案 |
|------|---------|
| CASE类型不匹配 | 用if-else替代条件参数 |
| 字段类型严格 | 状态字段使用VARCHAR(1) |
