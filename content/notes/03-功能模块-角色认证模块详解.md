## 一、模块概述

角色认证是面向用户的功能，用户可以主动申请某个角色，提交审批后由管理员审核。

### 1.1 与角色管理的关系

| 维度 | 角色管理 | 角色认证 |
|------|---------|---------|
| 使用者 | 管理员 | 普通用户 |
| 分配方式 | 直接分配 | 申请+审批 |
| 适用场景 | 后台管理 | 用户自助申请 |

**关联点**：
- 共享角色数据（`sys_role` 表）
- 审批通过后写入 `sys_user_role` 表

## 三、数据库设计

### 3.1 表结构

```sql
CREATE TABLE sys_role_auth (
    id VARCHAR(64) NOT NULL PRIMARY KEY,
    user_id VARCHAR(64) NOT NULL,           -- 申请人ID
    role_id VARCHAR(64) NOT NULL,           -- 申请的角色ID
    auth_content TEXT,                       -- 申请说明
    annex VARCHAR(500),                      -- 附件ID（多个用逗号分隔）
    status VARCHAR(1) DEFAULT '0',           -- 状态：0暂存 1待审批 2已通过 3已拒绝
    audit_result VARCHAR(1),                 -- 审批结果：0通过 1拒绝
    audit_remark VARCHAR(500),               -- 审批意见
    audit_time TIMESTAMP,                    -- 审批时间
    audit_by VARCHAR(64),                    -- 审批人ID
    submit_time TIMESTAMP,                   -- 提交时间
    -- 基础字段
    create_time TIMESTAMP,
    create_by VARCHAR(64),
    update_time TIMESTAMP,
    update_by VARCHAR(64),
    del_flag INTEGER DEFAULT 0
);
```

### 3.2 状态流转

## 五、核心业务逻辑

### 5.1 暂存认证申请

```java
@Override
public int saveRoleAuth(SysRoleAuthBo bo) {
    String userId = UserContext.getUserId();
    SysRoleAuth entity;

    if (StringUtils.isNotBlank(bo.getId())) {
        // 更新：校验状态和权限
        entity = getById(bo.getId());
        if (!STATUS_DRAFT.equals(entity.getStatus())) {
            throw new BizException("当前状态不允许修改");
        }
        if (!userId.equals(entity.getUserId())) {
            throw new BizException("无权操作此申请");
        }
    } else {
        // 新增
        entity = new SysRoleAuth();
        entity.setUserId(userId);
        entity.setStatus(STATUS_DRAFT);
    }

    entity.setRoleId(bo.getRoleId());
    entity.setAuthContent(bo.getAuthContent());
    entity.setAnnex(bo.getAnnex());

    return saveOrUpdate(entity) ? 1 : 0;
}
```

### 5.2 提交审批

```java
@Override
public int submitRoleAuth(SysRoleAuthBo bo) {
    String userId = UserContext.getUserId();

    // 校验是否已拥有该角色
    List<SysRole> userRoles = roleService.findRolesByUserId(userId);
    boolean hasRole = userRoles.stream().anyMatch(r -> r.getId().equals(bo.getRoleId()));
    if (hasRole) {
        throw new BizException("您已拥有该角色，无需申请");
    }

    // 校验是否有待审批的申请
    long count = count(new LambdaQueryWrapper<SysRoleAuth>()
            .eq(SysRoleAuth::getUserId, userId)
            .eq(SysRoleAuth::getRoleId, bo.getRoleId())
            .eq(SysRoleAuth::getStatus, STATUS_PENDING));
    if (count > 0) {
        throw new BizException("该角色已有待审批的申请");
    }

    entity.setStatus(STATUS_PENDING);
    entity.setSubmitTime(LocalDateTime.now());

    return saveOrUpdate(entity) ? 1 : 0;
}
```
